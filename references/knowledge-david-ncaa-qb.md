## 一、David 的训练体系核心 = Tudor Bompa 经典周期化

**不是 20 周闭环，是 52 周全年周期。** 之前基于截图的"20周"判断错误。

### 1.1 宏观结构（Annual Training Plan）

| 阶段 | 周数 | 强度 | 训练重点 |
|---|---|---|---|
| Active Recovery | ~2周 | — | 主动恢复 |
| Off-season（非赛季） | ~W3-16 | 65%→75% | Endurance → Hypertrophy → Basic Strength |
| Pre-season（赛前） | ~W17-28 | 80%→85% | Basic Strength → Strength-Power |
| In-season / Peaking（赛季） | ~W29-52 | 85%→90% | Strength-Power 维持 + 比赛 |

### 1.2 Mesocycle（中周期）规则

- **13个中周期 × 4周 = 52周**
- **每个中周期：3周加载 + 1周减量（recovery week）**
- **每8周一次测试周**（Testing Date）
- **Recovery 周强度降至 65-70%**，次数不变

### 1.3 强度-次数-组数的耦合曲线

从 Excel "周期训练计划"提取的实际参数：

| 阶段 | 组数 | 次数 | 强度%1RM | 适应目标 |
|---|---|---|---|---|
| Endurance | 5 | 12 | 65-70% | 肌耐力基础 |
| Hypertrophy | 4 | 10 | 70% | 肌肥大 |
| Basic Strength | 4 | 8 | 75% | 最大力量 |
| Strength-Power 早期 | 4 | 5-6 | 80-85% | 力量→速度转化 |
| Strength-Power 峰值 | 3 | 2-3 | 88-90% | 赛季爆发维持 |
| Deload 周 | 2 | 8-10 | 65% | 神经恢复 |

> **关键：强度上升 → 次数下降 → 组数小幅下降。** 这是 Bompa/Verkhoshansky 经典共轭/线性周期模型，不是 CrossFit 式混练。

## 二、一周四训标准模板（Off-season 强度日分配）

来源：Excel "橄榄球 一周四训 训练计划.xlsx" W1 处方。

| 日 | 强度系数 | 主题 | 核心内容 |
|---|---|---|---|
| **周一** | **88%** | Olympic + 上肢力量 | Power Clean / Hang Snatch / Power Jerk / 过头蹲 / 粗绳引体 / 耸肩 / 脖子 |
| **周二** | **68%** | 卧推日 + 单侧下肢 | 卧推 / 上斜 / 哑铃卧推 / 颈后深蹲 / 哑铃弓步 / GHR 北欧挺 / 侧平举 |
| **周四** | **86%** | Olympic 变体 + 跳跃 | Hang Snatch / Hang Clean / Split Jerk / Box Jump / 粗绳引体 / 附身划船 |
| **周五** | **71%** | 卧推金字塔 + 后链 | 卧推 10-8-6 / 上斜 / 哑铃卧推 / 颈后深蹲 / **前蹲** / **保加利亚** / **RDL** |

### 每日固定结构（David 模板）

```
1. MOVEMENT PREP 动态热身 + 专项康复（肩/膝/髋三选一）
2. HURDLE DRILL 栏架热身 OR FOOTLADDER 敏捷梯
3. MAIN LIFT 主项（Olympic 或 Squat/Bench）
4. SECONDARY LIFT 次项（同类变体）
5. ACCESSORY 辅助（上肢拉 / 后链 / 单侧）
6. PREHAB 预防（耸肩 / 脖子 / 二头三头）
7. CORE 核心
8. HIP SERIES 髋灵活度（每日必做）
```

### 强度百分比的用法

"周一 88%"**不是**每个动作都 88%，而是**主项相对本人该阶段计划强度的执行系数**。

举例（Strength-Power 期，计划强度 85%）：
- 周一 88% × 85% ≈ 75% 绝对负荷（主项）
- 周五 71% × 85% ≈ 60% 绝对负荷（volume 日）

这是经典的"heavy / light / medium / recovery"四日分布，保证 CNS 疲劳可恢复。

## 三、Week 11 实际处方（Off-season Basic Strength 样本）

来源：Excel "Week11训练计划.xlsx"，这是 David 给 Stan 实际开的单。

### 组合结构（PAP 对比法）

```
组合A（爆发力启动）
  - 哑铃深蹲跳 3×5
  - 药球下砸 + 深蹲跳 3×8
  - 高翻 75kg 5×5

组合B（主力量复合）
  - 杠铃背蹲 130kg 5×5
  - 哑铃平板卧推 35kg 5×5
  - 哑铃单侧附身划船 30kg 5×5

组合C（后链 + 拉）
  - 杠铃直腿硬拉 40kg 3×12
  - 引体向上 4×10
  - 器械二头弯举 3×12

辅助
  - 二/三头超级组 2×(10+10)
  - 核心：25 仰卧屈膝提髋 + 50 俄转 + 10 V 型起（无间歇）
```

> **解读：** 这是典型 Basic Strength 中段处方。组合A通过跳跃+下砸预激活 CNS，再做高翻（PAP 原理）；组合B 5×5 是力量黄金区间；组合C 高次数补薄弱环节。核心训练用固定 25-50-10 套路，避免每次重新设计。

## 四、脚踝三柱理论（Cal Dietz 体系，David 本土化）

来源：Excel "7.19 脚踝力量训练计划.xlsx"。

### 脚踝三柱训练

| 柱 | 训练动作 | 要点 |
|---|---|---|
| **外侧柱** | 单腿反复横跳 | 稳定踝外翻肌群 |
| **中外侧** | **Pogo 跳** | 前脚掌快速触地，膝锁定 |
| **内外中三柱** | 收腹跳 Bounding | 快速起跳，髋膝踝展直，保持最高 |

### 跳深进阶序列

1. 双脚跳深 → 单脚跳深（短跑专项）
2. 双腿侧向深跳与纵跳 → 单腿版本
3. **落地缓冲训练：软落地 + 脚趾抓毛巾**（经常被忽视的基础）

### 硬拉提踵变体

- 罗马尼亚硬拉提踵
- 单腿硬拉提踵（**脚后跟不落地，骨盆不旋转**）

> **这是"力量漏力 30%"的修复方案。** David 认为很多 QB/WR 力量练好了跑不快，根源在脚踝没有 Spring 储能。

## 五、视频截图提炼的训练哲学

### 5.1 "腰疼 3 步无痛训练"

1. **根因诊断不治症状** — 腰疼不是练腰，可能是髋屈肌紧张 + 核心失稳
2. **动作纠正** — 髋铰链模式重建
3. **渐进负荷** — 先无痛做动作，再加负荷

### 5.2 "NCAA 教练三大常见误区"

1. 把"累"当训练效果的衡量标准
2. 用**健美训练方法**练运动员（高次数、力竭、分肌群）
3. 忽视动作质量、过早堆重量

### 5.3 "越练越慢"的三个原因

1. 过度堆叠高力竭训练（CNS 过载）
2. 缺乏爆发速度训练，肌肉神经没有速度刺激
3. 柔韧性/动作幅度不足，力传递漏力

### 5.4 动态热身（全美冠军标准）

10-15 分钟，包含：
- 髋部打开（Hip Series 每日必做）
- 躯干启动
- 神经激活（敏捷梯 / 栏架）
- 专项模式预演

### 5.5 康复即训练

- 康复动作（肩膀/膝关节/髋）整合进每日热身
- 伤病员继续训练未受伤部位

### 4. 腰疼？别再盯着腰练了！
**类型：** 视频
**核心观点：** 腰是受害者不是原因；腰疼根因在于髋关节不灵活或胸椎僵硬导致腰椎被迫代偿。
**训练动作/协议（完整两步流程）：**
- **第一步·拉伸（拉髋不拉腰）：**
  - 髋屈肌拉伸（久坐8小时缩短8小时，站起来拉不开就把腰椎往前拽）
  - 髋关节内旋+外旋（90%的人完全忽略的方向；内外旋失衡腰椎变旋转轴心）
- **第二步·激活（叫醒沉睡肌肉）：**
  - 臀大肌：十个腰疼九个臀部不发力
  - 髋关节内外旋肌群：旋转稳定性不够腰椎扛不该扛的扭转力
  - 核心抗伸展/抗旋转/抗侧屈：核心不是用来动的，是用来锁住腰椎的
**适用对象：** 腰疼运动员/普通人，无需器械，每次15分钟。
**关键细节：** 单纯拉伸过一段时间又疼的原因：紧的松开了但弱的没练起来；必须拉伸完立即激活，趁神经抑制刚解除让肌肉重新学会发力。
**案例：** 全国冠军排球运动员Sloane，腰疼到无法正常训练，两个月后打满赛季零疼痛。
**赞/收：** 2.4万 | 日期：约2026年3月

### 5. 为啥你越练越慢？90%运动员训练方法错了
**类型：** 视频
**核心观点：** 特定适应性原则（SAID）——身体针对特定刺激产生特定适应，跨类别不迁移；天天练卧推投球速度不会变快，因神经募集路径完全不同。
**训练动作/协议：** NCAA橄榄球四分卫完整训练日：速度→爆发力→最大力量→辅助，四板块顺序有原因（神经系统最佳募集顺序）。
**适用对象：** 追求运动表现的运动员（特别是只做力量训练的）。
**关键细节：** 想跑快必须练冲刺；想投远必须练旋转爆发；想跳高必须练跳跃。力量是地基但地基不会自动变成楼，必须通过专项速度爆发力把力量转化为表现。20周计划包含四板块缺一不可。
**赞/收：** 495 | 日期：约2026年3月

### 6. 髋关节不灵活，腰和膝盖替你扛
**类型：** 视频
**核心观点：** 髋关节有6个功能方向；任何一个方向活动度不够，身体就会从腰椎或膝盖去借动。
**训练动作/协议（6方向完整打开）：**
| 方向 | 动作 | 参数 |
|---|---|---|
| ①屈曲 | 仰卧抱膝（拉臀大肌+腘绳肌） | 30秒×2 |
| ②伸展 | 半跪姿髋屈肌拉伸 | 30秒×2 |
| ③内收肌 | 抬高横向弓箭步 | 30秒×2 |
| ④外展 | 蝴蝶式拉伸 | 30秒×2 |
| ⑤外旋 | 90/90前腿方向 | 30秒×2 |
| ⑥内旋 | 90/90后腿方向 | 30秒×2 |
一轮不到4分钟，加到每次训练前热身。
**适用对象：** 橄榄球四分卫 + 所有有腰/膝代偿的运动员。
**关键细节：** 髋是球窝关节（天生用来旋转），膝是铰链关节（只能屈伸）；90%的人最缺内旋方向；外旋不足是投球转体的限制因素。
**赞/收：** 3158 | 日期：约2026年2月

### 7. 膝盖疼为什么要练屁股？光躺是躺不好的
**类型：** 视频
**核心观点：** 膝盖疼根源不在膝盖——髋关节不发力+臀部不激活+踝背屈不足，所有压力集中在髌腱。训练中产生的问题必须在训练中解决。
**训练动作/协议（4步完整康复）：**

## 十一、视频内容核心知识图谱（蒸馏总结）

### 11.1 David的诊断框架（贯穿所有21条视频）

**关节交替理论（Joint-by-Joint Approach）** 是所有方案的底层逻辑：
- 灵活关节（踝/髋/胸椎）出问题 → 稳定关节（膝/腰椎）被迫代偿 → 代偿关节产生疼痛
- 治疼痛要找上游，不治下游

**万能康复三步法（适用于所有关节）：**
1. 松解过紧的（泡沫轴/按压）
2. 拉伸紧张肌肉（先松再拉）
3. 激活被抑制的（弱侧/拮抗肌）
→ 整合进真实动作

### 11.2 运动表现训练三条线
| 训练线 | 内容 | 若缺失 |
|---|---|---|
| 力量线 | 深蹲/高翻/硬拉 | 基础不足，无法产生爆发力 |
| 速度线 | 冲刺/B式跳跃/C式跳跃 | 力量无法转化为速度 |
| 代谢线 | 代谢体能/赛季维持 | 后半段体力崩溃 |

**顺序原则（神经系统学习顺序）：** 速度→爆发力→最大力量→辅助
