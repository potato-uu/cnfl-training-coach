## Section 1：Tier System 核心——Total Body / Upper / Lower 每日分类逻辑

### 1.1 系统哲学

Tier System 的根本前提：**橄榄球是动作游戏，不是肌肉群游戏**。
Kenn 明确拒绝三个训练范式：纯健美（训练部位）、纯举重（技术导向）、纯力量举（最大力导向）。
替代方案叫 **Athletic Based Training（ABT）**——结合三者技术，但服务于"足球运动员"的目标。

> "Train Movements versus Training Body Parts. Football is a game of movement. Nobody cares how big your arms are."  
> ——Joe Kenn, 2002 Hammer Strength Clinic

### 1.2 三大动作类别定义

| 类别 | 英文 | 核心动作模式 | 代表动作 |
|---|---|---|---|
| **全身** | Total Body (T) | 爆发性髋伸展 + 肩部上提，多关节链接 | 悬挂抓举、高翻、Snatch变式、Power Pull、三重伸展组合 |
| **下肢** | Lower Body (L) | 膝主导 + 髋主导，单/双腿 | 深蹲变式、弓步、单腿蹲、臀腿举（GHR）、RDL |
| **上肢** | Upper Body (U) | 推与拉，水平/垂直 | 卧推、肩推、引体向上、划船 |

**关键逻辑：每节课包含三种类别，但优先级（Tier 1 > Tier 2 > Tier 3）每天轮换。**

### 1.3 每周旋转模板

一周三次（周一/三/五 或 周二/四/六）：

| 节次 | Tier 1（最高优先） | Tier 2（次优先） | Tier 3（辅助） |
|---|---|---|---|
| Session T（周一） | **Total Body** | Lower Body | Upper Body |
| Session L（周三） | **Lower Body** | Upper Body | Total Body |
| Session U（周五） | **Upper Body** | Total Body | Lower Body |

连续五周扩展版（Kenn 2002 原稿 Page 14）：

| 周 | Session T | Session L | Session U |
|---|---|---|---|
| 第1周 | T优先 / L次 / U辅 | L优先 / U次 / T辅 | U优先 / T次 / L辅 |
| 第2周 | L优先 / U次 / T辅 | U优先 / T次 / L辅 | T优先 / L次 / U辅 |
| 第3周 | U优先 / T次 / L辅 | T优先 / L次 / U辅 | L优先 / U次 / T辅 |
| 第4周 | 重复第1周 | 重复第1周 | 重复第1周 |
| 第5周 | 重复第2周 | 重复第2周 | 重复第2周 |

这种五周循环保证每个类别在每个 Session 里都轮流担任 Tier 1/2/3，实现真正的均衡发展。

### 1.4 Tier 扩展（3×3 / 3×5 / 3×7）

- **3×3（三级）**：每次3个动作，赛季中首选，重量室时间45分钟内
- **3×5（五级）**：每次5个动作，发展期常用，增加T4（第二个类别重复）和T5（第三个类别重复）
- **3×7（七级）**：七个动作，大训练量发展期，完整覆盖三类别各2遍+1

## Section 2：Major / Minor / Supplemental 动作分级（每日前3个位置怎么选）

### 2.1 四层动作分类体系

Kenn 将所有动作分为四类（书中更新版 vs. 2002 演讲版略有差异）：

| 级别 | 名称 | 选择标准 | 典型动作 | Tier 位置 |
|---|---|---|---|---|
| 1 | **Foundation（基础）** | 多关节杠铃动作，可测试最大力 | 深蹲、卧推、高翻、引体向上、硬拉 | Tier 1 首选 |
| 2 | **Supplemental（补充）** | Foundation 的变式，角度/范围变化 | 前深蹲、哑铃卧推、悬挂式高翻、斜面推 | Tier 1 或 2 |
| 3 | **Major Assistance（主要辅助）** | 多关节，非杠铃主导，拮抗或功能性 | 弓步、单腿蹲、划船变式、肩推变式 | Tier 2 或 3 |
| 4 | **Secondary Assistance（次要辅助）** | 单关节或靶肌肉隔离 | 三头肌、弯举、侧平举 | Tier 4/5 或 Blitz |

**实操逻辑：**
- Tier 1 = Foundation 动作 → 最高组数、最高强度
- Tier 2 = 第二类别的 Foundation 或 Supplemental → 中等组数、中等强度
- Tier 3 = 第三类别的 Supplemental 或 Major Assistance → 较低组数、较低强度
- Tier 4/5 = Assistance 辅助收尾

### 2.2 实际课程示例（来自 Kenn 2002 原稿 Page 25-27）

**Session T（周一）：**
- T1：全身 Foundation — 高翻 + 接铃（Full Pull and Catch）
- T2：下肢 Foundation — 深蹲变式（Back Squat）
- T3：上肢 Foundation — 平卧推（Flat Press / DB Bench）
- T4：全身 Supplemental — Hammer Jammer 推
- T5：下肢 Major Assistance — 弓步（Forward/Back Lunge）
- 后链：背伸展（Back Extension）
- 三联组：推/划（Pull/Row Tri-Set）

**Session L（周三）：**
- T1：下肢 Foundation — 深蹲变式（Back Squat）
- T2：上肢 Foundation — 平卧推（Chain Bench）
- T3：全身 Supplemental — 三重伸展（Clean Power Pull）
- T4：下肢 Major Assistance — 侧向台阶跨（Lateral Step Up）
- T5：上肢 Major Assistance — 哑铃垂直推
- 后链：早安式硬拉（Good Morning）

**Session U（周五）：**
- T1：上肢 Foundation — 卧推（Bench Press）
- T2：全身 Supplemental — 过顶（Split Jerk）
- T3：下肢 Foundation — 单腿蹲
- T4：上肢 Major Assistance — 站立肩推
- T5：全身 Major Assistance — 哑铃单臂抓举（DB SA Snatch）
- 后链：罗马尼亚硬拉（RDL）

### 2.3 位置专项动作差异（Page 29 原稿）

原稿 Page 29 实际课程表中明确标注 **BIGS（大体型）** 和 **SKILL（技巧）** 的动作差异：

| Tier | BIGS（OL/DL/LB） | SKILL（QB/WR/DB/RB） |
|---|---|---|
| T3 | Bear Squat（杠铃深蹲变式，双腿） | Split Squat（单腿分腿蹲，后腿台阶抬高） |
| T4（Session L） | Lateral Step Up（侧向台阶跨） | Lateral Lunge + Lateral Step Up |

这是 Kenn 书中 "Big / Big Skill / Skill" 三组分类的实操体现（见 Section 8）。

## Section 3：强度波动（Undulating）——Heavy / Medium / Light 日怎么分

### 3.1 三峰微周期（Three-Peaked Microcycle）

Kenn 2002 原稿 Page 13 明确提出两种微周期模型：

**传统单峰微周期（用于健美/力量举）：**
- 周一 Heavy → 周三 Moderate → 周五 Light

**Tier System 三峰微周期：**

| | Session T（周一） | Session L（周三） | Session U（周五） |
|---|---|---|---|
| **整体节次** | High Volume/Priority | Moderate Volume/Major | Low Volume/Minor |
| **T1强度** | High/Priority（最重） | High/Priority（最重） | High/Priority（最重） |
| **T2强度** | Moderate/Major | Moderate/Major | Moderate/Major |
| **T3强度** | Low/Minor | Low/Minor | Low/Minor |

**核心差异：每个 Session 内部都有 Heavy-Medium-Light 层级，但每天的"最重"动作类别不同。**
不存在"整节课轻日"——每节课都有 Tier 1 高强度，只是类别在轮换。

### 3.2 强度努力类型（Effort Types）

| 类型 | 英文 | 负荷% | 次数 | 目标 |
|---|---|---|---|---|
| 最大力努力 | Max Effort (ME) | >80% 1RM | 1-5次 | 绝对力量 |
| 动态努力 | Dynamic Effort (DE) | 40-60% 1RM | 1-3次（最大速度） | 爆发力/Rate of Force Development |
| 重复努力 | Repeat/Sub-Max Effort (RE/SME) | 60-80% 1RM | 1-10次 | 肌肉增生/技术强化 |

**爆发力重点周（Speed/Power Focus）：**
- Session T：DE-全身 + ME-下肢 + RE-上肢
- Session L：DE-下肢 + ME-上肢 + RE-全身
- Session U：DE-上肢 + ME-全身 + RE-下肢

**力量重点周：**
- Session T：ME-全身 + DE-下肢 + RE-上肢

### 3.3 传统 Heavy/Moderate/Light 配置（书中版本）

Ken Gunter 播客中 Kenn 提到"Traditional Template"：
- 5×5 at 85% → 75% → 65% 对应 Heavy/Moderate/Light 三级

**实际赛季示例（原稿 Page 29，Spring II/Summer 发展期第7-8周）：**

以悬挂高翻（Hang Clean，1RM=300lb）为例的 Cluster Set 方案：

| 周 | 组次结构 | 强度% | 重量 |
|---|---|---|---|
| 第7周 | 激活 55%×5 → 61%×2 → 70%×1 → 76%×1 | 然后 Cluster | 170 → 190 → 220 → 235 |
| 第7周 Cluster | 82%×3次×6组（组内每次20秒休息，组间队友轮换） | 82% | 255 lb |
| 第8周 | 激活 55%×5 → 64%×2 → 73%×1 → 79%×1 | 然后 Cluster | 170 → 200 → 230 → 245 |
| 第8周 Cluster | 85%×3次×6组 | 85% | 265 lb |

同期深蹲（1RM=500lb）Cluster 方案：

| 周 | 组次 | 强度% | 重量 |
|---|---|---|---|
| 第7周 | 激活后 82%×5次×6组 Cluster | 82% | 255 lb |
| 第8周 | 激活后 85%×4次×6组 Cluster | 85% | 265 lb |

同期卧推（1RM=400lb）方案：

| 周 | 组次 | 强度% | 重量 |
|---|---|---|---|
| 第7周 | 激活后 82%×5次×6组 Cluster | 82% | 340 lb |
| 第8周 | 激活后 85%×4次×6组 Cluster | 85% | 355 lb |

**Cluster Set 说明：** 组内每次之间休息20秒（置铃，与队友轮换节奏），组间完全休息。这是 Kenn 在 NFL/大学训练中大量使用的核心组次工具。

## Section 4：赛季阶段划分

### 4.1 Kenn 的年度训练周期（American Football Calendar）

```
[1月-3月]    Postseason + Winter Off-Season
             目标：解剖适应（AA Phase / Block Zero）、恢复、基础容量建立
             频率：3×/周，3×3 → 3×5 Tier

[3月-4月]    Spring Ball（春训/春季赛季）
             目标：技术+力量并行，3×5 Tier
             注：NCAA 规定春训15次练习，力量训练配合技术课

[5月-8月]    Summer Offseason（暑期非赛季）
             目标：最大化力量/爆发力峰值
             频率：3×/周，3×5 或 3×7 Tier
             强度：最高阶段，Cluster Sets，Peaking Blocks

[8月]        Preseason Camp（季前营）
             目标：保持力量、适应比赛频率、不增加额外疲劳
             频率：降至2×/周，3×3 Tier
             重量室时间：≤45分钟/次

[9月-1月]    In-Season（赛季中）
             目标：维持峰值力量/爆发力，不让增益丢失
             频率：2×/周（NFL 通常周日+周三）
             结构：3×3 Tier，减少 Tier 数量

[1月后]      Postseason / Bowl Games → 重回 Offseason
```

### 4.2 发展期 vs 竞技期的核心区别

| 参数 | Developmental（发展期） | Competitive（竞技期） |
|---|---|---|
| 每周举次 | 3次 | 2次 |
| Tier 深度 | 3×5 或 3×7 | 3×3 |
| 每节课时长 | 60-90分钟 | ≤45分钟 |
| 主要目标 | 力量增长 | 维持+疲劳管理 |
| 后链工作 | 完整（GHR/RDL/Good Morning） | 精简（2-3组维持） |

## Section 5：4周块的具体组次强度

### 5.1 Kenn 的中周期块结构（Mesocycle，3-4周）

Kenn 的典型中周期为**3-4周一块**，以 Prilepin 图表为强度框架基础。

**通用4周块（书中描述，基于 VanDyke/EliteFTS 交叉验证）：**

| 周次 | 训练主题 | 代表性组次方案 | 强度范围 | 备注 |
|---|---|---|---|---|
| 第1周 | 容量/解剖适应 | 3×8 - 4×8 | 60-70% 1RM | 高次数，技术强化 |
| 第2周 | 肌肉增生/容量 | 4×6 - 5×6 | 70-75% 1RM | 容量峰值 |
| 第3周 | 力量发展 | 5×5 - 4×5 | 75-82% 1RM | 力量导向 |
| 第4周 | 峰值力量/减量 | 6×3 - 4×3 | 82-90%+ 1RM | 低容量高强度，或测试 |

**进阶发展期 Cluster 块（来自原稿 Page 29，Summer Phase）：**

- 激活爬坡组：55%×5 → 61%/64%×2 → 70%/73%×1 → 76%/79%×1
- 主工作组：82-85% × 3-5次 × 6组 Cluster（20秒组内间歇）

### 5.2 基础力量块与爆发力块切换逻辑

Kenn 整合了 Westside Barbell、传统线性、HIT 三个体系：

| 训练块 | 主导方法 | 典型方案 |
|---|---|---|
| 力量积累块（早期非赛季） | 线性渐进 + RE/SME努力 | 3×10 → 4×8 → 5×5 → 6×3 |
| 爆发力峰值块（暑期后段） | ME + DE + Cluster | 激活爬坡 + 82-88%×3-6组 |
| 赛前保持块（Camp前） | 降容量保强度 | 3×3 - 4×3，强度不降 |
| 赛季维持块 | 最小有效剂量 | 2次/周，各 Tier 1 Foundation 3-4组 |

### 5.3 Prilepin 图表在 Tier System 中的应用

Kenn 明确将 Prilepin 图表（苏联举重负荷量化工具）整合到 Tier 1 Foundation 动作的组次选择中：

| 强度% | 每组次数 | 每节课最优总次 | 允许范围 |
|---|---|---|---|
| 55-65% | 3-6次 | 24次 | 18-30 |
| 70-75% | 3-6次 | 18次 | 12-24 |
| 80-85% | 2-4次 | 15次 | 10-20 |
| 90%+ | 1-2次 | 7次 | 4-10 |

（注：以上为标准 Prilepin 表格；Kenn 在书中直接引用此框架）

## Section 6：Olympic Lifts 在 Tier System 里的位置

### 6.1 核心地位

> "The Olympic Movements and their variations are the key to developing total body strength. These exercises train the body as a complete unit."  
> ——Joe Kenn, 2002 Hammer Strength Clinic

奥林匹克举重动作是 Tier System **Total Body（全身）** 类别的首选 Foundation 动作，始终在 Tier 1 或 Tier 2 出现。

### 6.2 奥举目标的三重功能

1. **踝-膝-髋三重伸展（Triple Extension）**——所有运动中最关键的爆发力动作
2. **协调性与身体意识**——在最大速度下控制身体
3. **功能性全身力量**——而非孤立肌肉

### 6.3 Tier System 内的奥举分类（原稿 Page 16）

**拉类（Pulling Movements）：**
- 完整拉接（Full Pull and Catch）
- 悬挂式高翻（Hang Clean）
- 地面高翻（Clean from the Deck）
- 三重伸展（Triple Extension）
- 高翻拉力（Clean Power Pull）
- 悬挂推举（Hang Push Jerk）

**推类（Pushing Movements，奥举变式）：**
- Hammer Jammer 推
- 上推（Push Press）
- 推举（Jerks）
- 分腿挺举（Split Jerk）
- 悬挂蹲举（Push Jerk）

**组合动作（Combination Movements）：**
- 悬挂高翻 + 前深蹲（Hang Clean to Front Squat）
- 抓举 + 过顶深蹲（Power Snatch to Overhead Squat）
- 哑铃高翻 + 推举（DB Clean to Push Press）

### 6.4 Olympic Lift 在课程中的位置规则

- 当 Session 为 T 优先（Session T）→ 奥举在 **Tier 1**，最大力/爆发力主导
- 当 Session 为 L 或 U 优先 → 奥举降至 **Tier 2 或 3**，作为补充
- 原稿实际课程（Page 25-27）：
  - Session T：T1 = 高翻，T4 = Hammer Jammer，T5（非全身）
  - Session L：T3 = Clean Power Pull（全身降至 Tier 3）
  - Session U：T2 = Split Jerk，T5 = DB SA Snatch

### 6.5 不同运动员水平的奥举处理

- **初级运动员**：从 Hang Clean（悬挂式）开始，避免地面拉（技术要求更高）
- **中级**：加入 Power Pull、Clean Power Pull 变式
- **高级**：完整动作 + Cluster Sets + 大强度（82-88% 1RM）

## Section 7：赛季中（Inseason）维持方案——两次举起保峰值

### 7.1 核心原则

Kenn 的赛季内原则：**以最小剂量维持非赛季的力量峰值**，不追求新增长，不制造额外 CNS 疲劳干扰比赛表现。

> "During the competitive stage the 3×3 Tier is a beneficial approach for football players. As a strength coach, there are limitations on how much time we have with athletes."

### 7.2 赛季内结构（3×3 Tier）

**每周举重2次**（典型NFL：周日比赛后恢复训练 + 周三强度保持）：

| 日次 | 内容 | 强度 | 时间 |
|---|---|---|---|
| 举重1（比赛后1-2天） | 3×3 Tier，恢复导向，较低强度 | 60-70% | ≤45分钟 |
| 举重2（比赛前3天） | 3×3 Tier，维持导向，保持强度 | 75-85% | ≤45分钟 |

**每次课结构（3×3 Tier 赛季版）：**
- Tier 1：Foundation 动作（该日类别优先），3-5组
- Tier 2：第二类别 Supplemental，3-4组
- Tier 3：第三类别 Assistance，2-3组
- 后链（Posterior Chain）：必做，1-2个动作，2-3组

### 7.3 Readiness / wRap Up 工作

每节课额外加入**最小可行恢复组合**（≥1个动作/部位）：
- 颈部（手动阻力/弹力带）
- 核心/Power Zone
- 后链（GHR / Pull Through / RDL）
- 后三角肌/后肩

### 7.4 Blitz 附加训练

Blitz Package = 独立于3次主训练的高强度短训，每次**≤20分钟**，每周≤5次（主训3小时后进行），**自愿参加**。

目标：提升工作容量、弱点补充、预防受伤、恢复促进。
赛季内可继续使用 Blitz 处理个人专项弱点（位置特异性工作）。

## Section 8：位置专项差异（Skill / Big Skill / Big）如何调整

### 8.1 Kenn 的三组分类

Kenn 将橄榄球位置分为三类：

| 组别 | 包含位置 | 特点 |
|---|---|---|
| **SKILL（技巧位）** | QB、WR、DB、RB | 速度/变向主导，体重较轻，奥举敏感性高 |
| **BIG SKILL（大技巧位）** | TE、LB、S（安全卫） | 力量+速度兼顾，中间体型 |
| **BIG（大体型位）** | OL（进攻锋线）、DL（防守锋线） | 力量/块面主导，体重最重，单腿动作受限 |

### 8.2 动作选择差异（来自原稿 Page 29 实际训练表）

**下肢类别（Lower Body）T3/T4 位置差异：**

| Tier | BIG | SKILL |
|---|---|---|
| T3（Session T/U） | Bear Squat（双腿，平行） | Split Squat / Bulgarian Split Squat（单腿后脚台阶） |
| T4（Session L） | Lateral Step Up（侧向台阶跨） | Lateral Lunge + Lateral Step Up 组合 |
| T5（Session T） | Combo Lunge（哑铃组合弓步，分钟为计） | Band Hip Extension（弹力带髋伸展，单腿） |

### 8.3 强度/容量差异原则

- **BIG（锋线）**：更高绝对重量，双腿动作优先，奥举多用 Power Pull / Push Press 而非完整接铃
- **SKILL（技巧位）**：奥举完整动作优先（接铃技术），单腿动作占比更高，爆发力/反应力量训练权重更大
- **BIG SKILL**：介于两者之间，根据个人需求调整

### 8.4 年度重点差异

| 阶段 | BIG 重点 | SKILL 重点 |
|---|---|---|
| 非赛季早期 | 基础力量积累（3×8 - 5×5） | 协调力 + 基础力量并行 |
| 非赛季中期 | 最大力峰值（5×3 - 6×3） | 爆发力 + 反应速度 |
| 赛季内 | 维持绝对力量 | 维持爆发力/弹性力量 |

### 8.5 器械工具偏好

- **Hammer Jammer**：OL/DL首选全身爆发力工具（模拟阻击/推挡动作）
- **单腿动作**：SKILL 位球员获益更大（单腿力量直接转化加速/变向）
- **弓步矩阵（各方向）**：所有位置，但 SKILL 占比更高

## Section 9：与 Bompa / Triphasic 的异同

### 9.1 与 Tudor Bompa（线性/传统周期化）的比较

| 维度 | Bompa 传统周期化 | Kenn Tier System |
|---|---|---|
| **核心组织原则** | 大周期 → 中周期 → 小周期线性递进 | 以动作类别旋转为骨架，强度在框架内波动 |
| **每日安排** | 力量日 / 爆发力日 / 恢复日分离 | 每日均含三类别，每日内部有 Heavy-Moderate-Light 层级 |
| **运动专项性** | 通用到专项线性过渡 | 始终以运动员身份训练（Athletic Based Training），无"纯通用"期 |
| **测试节点** | 中/大周期末测试 | Foundation 动作中周期末测试（RM测试） |
| **适合对象** | 个人运动项目、Olympic track | 团队运动，尤其是 American Football |
| **赛季内** | 大幅减量，保守 | 结构化2×/周维持，保强度 |
| **相同点** | 均认可周期化必要性；均使用 Prilepin 强度分区 | - |

### 9.2 与 Cal Dietz（Triphasic Training）的比较

| 维度 | Triphasic（Cal Dietz） | Kenn Tier System |
|---|---|---|
| **核心创新** | 将动作拆为离心/等长/向心三相，分别周期化训练 | 动作类别旋转 + 努力类型（ME/DE/RE）分配 |
| **每日节奏** | 纯离心周期 → 纯等长周期 → 向心/爆发力周期（各1-3周） | 每次课内部均包含三类别，不单独分离某相 |
| **Olympic Lifts** | 有限（Dietz 更多用 French Contrast 法式对比） | 核心地位（Total Body 类别首选） |
| **位置差异** | 提供 Block/Power/Speed 分级，但不按位置区分 | 明确 BIG / BIG SKILL / SKILL 三组，动作有差异 |
| **赛季周期** | 单峰周期向峰值推进，明确峰值日期 | 赛季内2×维持，不刻意设计"峰值日" |
| **法式对比** | 核心工具（RM后立即配合反应跳） | 未作核心工具，但 Cluster Sets 有类似神经激活效应 |
| **科学引用** | 大量肌电/神经科学文献 | 实战经验为主，引用 Prilepin 数据 |
| **相同点** | 均重视爆发力三重伸展；均用奥举变式；均3日/周结构 | - |

### 9.3 与 Westside Barbell（Louie Simmons）的比较

| 维度 | Westside Conjugate | Kenn Tier System |
|---|---|---|
| **ME/DE 分离** | 严格ME日 vs DE日（周一/三/五/六四天） | 同一日内包含 ME + DE + RE 三种努力 |
| **变异性** | 每周换 ME 动作，防止停滞 | 4-5周同一 Foundation 动作，保证技术积累 |
| **Olympic Lifts** | 基本不用（认为技术要求干扰训练） | 核心训练工具 |
| **运动员适用** | 原设计用于 PL，后推广至运动员 | 专为团队运动员（特别是美式橄榄球）设计 |
| **Kenn 的引用** | Kenn 明确将 Westside 列为五个来源之一（原稿 Page 12） | 吸收了 ME/DE 逻辑，但整合到 Tier 框架 |

### 9.4 三方法核心差异一句话总结

- **Bompa**：按时间线性推进专项化，从通用到专项
- **Triphasic**：按肌肉收缩相分离训练，神经系统最优化
- **Kenn Tier**：按动作类别旋转优先级，在每日内部完成三层强度分配，**始终以运动员身份训练**
