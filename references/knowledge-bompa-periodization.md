## Section 1：核心公理（Training Axioms）

### 1.1 适应原理（Adaptation）

- 训练 = 系统性暴露于不同 volume/intensity 的应激（stressor）
- 适应三阶段（GAS/Selye）：**警觉期（Alarm）→ 抵抗期（Resistance）→ 衰竭期（Exhaustion）**
- 超量恢复（Supercompensation）= 消除疲劳后反弹至新的 homeostasis 高点
- **疲劳与训练收益比 = 3:1**（Zatsiorsky & Kraemer）：疲劳持续时间仅是适应提升的 1/3
- 阈值突破原则：必须给予超出当前适应阈值的刺激才能进步；低于阈值→停滞；高于极限→过度训练
- 累积训练效应（Cumulative Training Effect）：短期内"看不见"，某周突然爆发

### 1.2 超量恢复时间窗（Supercompensation Window）

- 糖原补充：**20-24 小时**（若碳水不足或肌肉损伤严重则延长）
- 运动后 2 小时内是糖原合成速率最高窗口
- EPOC（运动后耗氧量升高）持续：**24-38 小时**
- 静息代谢率升高：**15-48 小时**
- 磷酸肌酸（PCr）恢复：高强度运动中 10 秒内下降 50%，30 秒后贡献极少
- 高强度训练间隔若足够，体内糖原和磷酸系统完全恢复后会进入超量补偿状态
- 训练频率建议：**不要每天做 maximal attempts** → 大幅减少适应能力（Zatsiorsky）

### 1.3 训练因素（Training Factors）

核心三要素：
| 要素 | 定义 | 量化方式 |
|---|---|---|
| **Volume（量）** | 总工作量（距离/重复次数/总重量/时长） | 公吨（metric ton）= sets × reps × load ÷ 1000 |
| **Intensity（强度）** | 训练质量（% 1RM / 速度 / 功率输出） | % of 1RM，心率区间，功率瓦数 |
| **Frequency（频率）** | 每周训练次数 / 密度 | 次/周，绝对密度 vs 相对密度 |

**Volume 与 Intensity 反比关系**：备赛期初 → 高 volume / 低 intensity；竞技期临近 → 低 volume / 高 intensity（taper）

### 1.4 强度分区（Intensity Zones）

#### 力量训练强度（% 1RM）
| 区间 | % 1RM | 肌肉动作 | 用途 |
|---|---|---|---|
| 超极限 | >100% | 离心 / 等长 | 顶级运动员，谨慎使用 |
| 极限 | 90-100% | 向心/离心 | 最大力量 |
| 重载 | 80-90% | 向心/离心 | 最大力量 |
| 中重 | 70-80% | 向心/离心 | 力量/爆发力过渡 |
| 中等 | 50-70% | 向心/离心 | 力量/爆发力 |
| 低 | 20-50% | 向心/离心 | 肌肉耐力/技术 |

**weightlifter 实际训练强度分布**：低强度 8% / 中等 59% / 重载 26% / 极限 7%

#### 有氧耐力区间
| 区间 | %最大心率 | %VO2max | 用途 |
|---|---|---|---|
| 活动恢复 | <60% | <40% | 再生 |

## Section 2：年度周期化模型（Annual Plan）

### 2.1 三大训练阶段

#### 备赛期（Preparatory Phase）
- **时长**：占年度训练的大部分时间（见表 5.1）
- 两个子阶段：
  - **一般备赛（General Preparation，GP）**：发展多功能运动能力，高 volume / 低 intensity；新赛季开始时从 30-50% 最大工作能力起步
  - **专项备赛（Specific Preparation，SP）**：向专项靠拢，技战术+专项体能，intensity 逐步提升

**GP vs SP 体积比**：初学者 GP:SP ≈ 2:1；精英运动员 GP:SP 缩短但 SP 密度更高

#### 竞技期（Competitive Phase）
- **时长**：
  - 团队项目（联赛制）：8-10 个月
  - 个人项目：更灵活
- 三个子阶段：
  1. **赛前准备（Precompetitive）**：参加次要比赛作为评估工具；短减量 3-7 天
  2. **主竞技期（Main Competition）**：维持力量/爆发力/耐力；减少训练量，维持强度
  3. 不能每场比赛都峰值

#### 过渡期（Transition Phase）
- **时长**：≤4 周（超过 4 周开始脱训效应）
- 目的：消除生理和心理疲劳，为下一周期准备
- 策略：主动休息（active rest）优于被动休息；体积降低 50-60%；强度也降低
- 超过 4 周：开始明显丢失力量和爆发力

### 2.2 年度计划类型（Annual Plan Types）

#### 单周期（Monocycle）— TABLE 5.1 数据
| 阶段 | 周数 |
|---|---|
| 备赛期 | ≥28 周 |
| 竞技期 | 12-22 周 |
| 过渡期 | 4 周 |
| **总计** | **52 周** |

- 适用：新手/青少年；耐力项目（北欧滑雪、赛艇、自行车、长跑）；季节性项目
- 特点：最长备赛期 → 最充分的适应基础

#### 双周期（Bi-cycle）
| 阶段 | 周数 |
|---|---|
| 备赛期 1 | 12-16 周（约 3 个月，含 GP+SP） |
| 竞技期 1 | 6-12 周 |
| 过渡期 1 | 1-4 周 |
| 备赛期 2 | 较短 |
| 竞技期 2 | 较短 |
| 过渡期 2 | 1-4 周 |
| **每周期** | **26 周** |

- 适用：可参加全国锦标赛的高级运动员；有冬季+夏季两个主要赛事的运动员

#### 三周期（Tri-cycle）
| 阶段 | 周数 |
|---|---|
| 周期 1 备赛期（冬季） | 12-20 周 |
| 周期 1 竞技期（冬季） | 4-6 周 |
| 周期 1 过渡期 | 1-2 周 |
| 周期 2 备赛期（春/夏） | 10-12 周 |
| 周期 2 竞技期 | 8-12 周 |
| 周期 2 过渡期 | 1-2 周 |
| 周期 3 备赛期（夏季） | 2-4 周 |
| 周期 3 竞技期 | 2-4 周 |
| 周期 3 过渡期 | 3-6 周 |

- 适用：国际级运动员；有冬锦赛→全国赛→世界锦标赛（或奥运会）三峰

#### 多峰计划（Multiple-Peak）
| 参数 | 数值 |
|---|---|
| 每周期总周数 | 11-16 周 |
| 备赛期 | 9-12 周 |
| 竞技期 | 1-2 周 |
| 过渡期 | 1-2 周 |

- 适用：训练背景 4-6 年以上、竞技次数极多的顶级运动员（典型：网球）
- 风险：无法充分达到每个生物运动能力的最高水平；不适合青少年

### 2.3 Volume/Intensity/Stress 曲线规律

- 备赛期初：volume 高，intensity 低，stress 由 volume 主导
- 备赛期末：volume 下降，intensity 上升
- 竞技期：volume 最低，intensity 维持高，stress 由 intensity 主导
- 过渡期：两者均低

## Section 3：三级周期定义与设计

### 3.1 宏观周期 / 大周期（Macrocycle）

- **定义**：年度计划的大分段，通常 2-6 周，指向一个主要竞技目标
- **备赛期宏观周期**：结构更灵活，负荷可以递增/递减/阶梯/平稳
- **竞技期宏观周期**：由比赛日历决定；应在末尾包含 unloading microcycle
- **2 周超负荷 → 需 2 周卸载**（Halson 等研究：2 周高负荷后，需 2 周卸载恢复到预负荷水平）
- 宏观周期内负荷波动四种模式：递增（ascending）/ 递减（descending）/ 阶梯（step）/ 平稳（flat）

### 3.2 中观周期 / 中周期（Mesocycle）

- Bompa 未明确定义单独的 mesocycle，但章节层次上对应 3-6 周的连续训练块
- 强度相位序列（MxS phase 为例）：3+1 模式（3 个递增负荷 microcycle + 1 个恢复 microcycle）

### 3.3 微观周期（Microcycle）

- **定义**：3-7 天，传统为 1 周；年度计划最重要、最功能性的单元
- 关键设计原则：负荷波动（volume + intensity 交替）以防止过度训练

#### 强度分级（训练需求）
| 等级 | % 最大能力 |
|---|---|
| 极高 | 90-100% |
| 高 | 80-90% |
| 中 | 70-80% |
| 低 | 50-70% |
| 极低 | <50% |
| 恢复 | 0（休息） |

#### Microcycle 类型

| 类型 | 特征 | 典型安排 |
|---|---|---|
| **发展型（Developmental）** | 主要目标是提升能力 | 1 峰（单高点）或 2 峰（双高点） |
| **单峰 1-peak** | 周中单次高强度 | 周二或三达峰；周末较低 |
| **双峰 2-peak** | 周内两次高强度峰值 | 周一+周五；适合竞技期 |
| **3峰 3-peak** | 三次高强度，需较高训练年龄 | 高级运动员 |
| **冲击型（Shock）** | 极高负荷，超负荷刺激 | 短期 1-2 周；须后接恢复周 |
| **卸载/恢复型（Unloading/Peaking）** | 减量以超量恢复 | 负荷降 40-60%；放在大周期末 |
| **恢复再生型（Recovery/Regeneration）** | 消除积累疲劳 | 低强度；放在长竞技期中间；2 周高负荷后需要 |
| **竞赛型（Competition）** | 比赛为核心，训练为辅 | 赛前 2-3 天减量；赛后再生 |

#### 3+1 负荷模式（Bompa 标准）
- 3 个负荷递增的 microcycle + 1 个卸载 microcycle = 1 个 macrocycle
- 4 周 macrocycle 典型：周 1 中→周 2 高→周 3 极高→周 4 卸载

#### 微周期交替原则（能量系统）
| 日类型 | 主要能量系统 | 内容 |
|---|---|---|
| 无氧磷酸系日 | ATP-PCr | 技术（1-10s）/ 加速 / 最大力量爆发力 |
| 无氧乳酸日 | 快速糖酵解 | 技术（10-60s）/ 速度耐力 / 爆发力耐力 |
| 有氧日 | 氧化系统 | 长时技术（>60s）/ 长中距战术 / 耐力 |

## Section 4：各训练块参数（Phases of Strength）

### 力量周期化七个阶段序列

**序列**：解剖适应（AA）→ 肌肥大（Hypertrophy）→ 最大力量（MxS）→ 转化（Conversion）→ 维持（Maintenance）→ 停止（Cessation）→ 补偿（Compensation）

---

### Phase 1：解剖适应（Anatomical Adaptation，AA）

**目的**：强化肌腱/韧带/关节；增加骨密度和结缔组织；不以增肌或立即超负荷为目标

**时长**：≥3 周；通常 4-8 周（视运动员训练背景）

**关键参数**：
- 每组时间张力（TUT）：**30-70 秒**（无氧乳酸系统为主能量系统）
- 向心速度：爆发性
- 离心速度：**3-5 秒/次**（慢离心为主）
- 氢离子（来自乳酸）刺激生长激素 → 促胶原合成
- 强调拮抗肌平衡；优先单侧动作

**处方范围**（基于书中描述）：
- 强度：50-70% 1RM（中低区间）
- 组数：3-5 组
- 次数：10-20 次
- 休息：60-90 秒

---

### Phase 2：肌肥大（Hypertrophy）

**目的**：增加肌肉横截面积；提升高能底物储量和酶浓度

**适用运动员**：需要增重的运动员（田径投掷、橄榄球前锋）；初学者

**与健美的区别**：
- 运动员肥大：更低平均次数 / 更重负荷 / 更长组间休息
- 运动员始终尝试以最快速度举起重量（concentric 爆发）
- 目标 fast-twitch 纤维肥大（IIa/IIx）

**处方范围**：
- 强度：**65-85% 1RM**（中等到中重）
- 组数：**3-5 组**
- 次数：**6-12 次**
- 休息：**60-90 秒**（短休息 → 更高 GH 分泌）
- TUT：30-60 秒/组

---

### Phase 3：最大力量（Maximum Strength，MxS）

**目的**：提升神经系统募集最大快缩肌纤维的能力；提高绝对力量

**时长**：≥4 周；典型 4-8 周；分 MxS-I（间肌协调）和 MxS-II（肌内协调）两阶段

**MxS-I（间肌协调 / 技术力量）**：
- 强度：**40-80% 1RM**（尤其 70-75% → 75-80%）
- 目的：学习动作，多组爆发性重复

**MxS-II（肌内协调 / 快缩纤维募集）**：
- 强度：**80-90% 1RM**（MxS-II 阶段）
- 目的：强制募集大运动单元

**MxS 处方范围**：
- 强度：**80-90% 1RM**（核心工作组）
- 组数：**4-8 组**
- 次数：**1-5 次**
- 休息：**3-5 分钟**（全恢复）
- 爆发向心；离心受控

**相位顺序（Macrocycle 内）**：
```
AA → MxS-I (3+1) → MxS-I (3+1) → MxS-II (3+1) → Power (2+1) → Power (2+1)
70-75%         75-80%              85-90%
```

---

### Phase 4：转化（Conversion to Specific Strength）

三条转化路径，根据运动项目需求选择：

#### 4a. 转化为爆发力（Conversion to Power）

**目的**：将最大力量储备转化为高速力量输出（RFD）

**时长**：≥4 周

**处方**：
- 强度：**40-80% 1RM**（中等负荷，**意图以最快速度移动**）
- 组数：**3-5 组**
- 次数：**3-6 次**（每次全力爆发）
- 休息：**3-5 分钟**（全恢复保证质量）
- 方法：弹道训练（ballistic）/ 上下肢增强式（plyometrics）/ 举重衍生动作

**Cluster Set 方法（Power Training）**：
- 模式 1（Olympic lifts）：2×10 reps，组内每rep 间休息 30s，组间休息 180s（2×(10×30s)/3min）
- 模式 2（力量动作）：3×5 reps，组内休息 1-3s，组间 3min（3×(5×1-3s)/3min）

#### 4b. 转化为爆发力耐力（Power Endurance）

- 强度：中等（50-70% 1RM），速度快
- 次数：较多（8-15 次），时间较长的组

#### 4c. 转化为肌肉耐力（Muscle Endurance）

- 短时肌肉耐力（无氧乳酸主导）：15-30 次
- 中时（有氧为主）：30-50 次
- 长时（有氧主导）：50+ 次，极端情况下需要 400 次/组
- 必须与代谢训练同步进行

---

### Phase 5：维持（Maintenance）

**目的**：在竞技赛季期间保留所有之前建立的适应

**处方**：
- 每周至少 **1 次** 专项力量训练
- 强度不降（维持 MxS 和 Power 强度）
- 减少 volume 即可
- 停止力量训练 → 竞技期逐渐性能下降

---

### Phase 6：停止期（Cessation）

- 在主要比赛前 **3-14 天** 结束力量训练
- 女性：停止早至赛前 3 天
- 速度/爆发力项目（alactic）：1-2 周停训可通过 IIx 纤维"反弹"提升爆发速度
- 长耐力项目：可在赛前 2 周停训
- 体重较重运动员：尽早停止（残余疲劳持续更长）

---

### Phase 7：补偿/过渡（Compensation/Transition）

- 时长：≤4 周（超过则脱训）
- 体积降低 **50-60%**
- 强度也显著降低
- 重点：拮抗肌、稳定肌、可能存在失衡的肌肉
- **不做任何力量训练的运动员会失去肌肉大小和相当程度的爆发力**（文献支持）

---

## Section 5：峰值设计（Peaking & Tapering）

### 5.1 减量协议（Taper Protocol）

**目的**：降低训练诱导的疲劳，提升 readiness（准备状态）；平均提升表现约 **3%**

**标准 Taper 时长**：**8-14 天**
- 若赛前训练负荷极高 → 需更长减量（或用快速指数减量策略）

#### 核心参数（TABLE 9.6）

| 参数 | 策略 | 效果 |
|---|---|---|
| **Volume（量）** | 减少总距离/时长 **40-60%**；减少重复次数；延长休息至完全恢复 | 生理超量补偿；神经系统就绪；能量储备充盈 |
| **Intensity（强度）** | 爆发力运动：降低 **5-10%**；耐力运动：降低 **20-30%**（第 1 周）；赛前数天重新**提高强度** | 维持神经肌肉适应 |
| **Frequency（频率）** | 维持在 pretaper 频率的 **80% 以上** | 维持适应质量 |

若赛前 volume 极高，volume 可减少 **60-90%**

#### 减量类型
- **渐进减量（Progressive Taper）**：系统性逐步减少负荷
  - 线性减量（Linear）：较高平均 volume，最终 volume 最低
  - 快速指数减量（Fast Exponential）：早期大幅降量；peak 点 volume 最低
- **非渐进减量（Non-progressive/Step Taper）**：标准化阶梯式降量
  - 平均 volume 更低；最终 volume 较高

#### 减量期 Microcycle 模式
- 主要比赛：减少 microcycle 内训练负荷 **40-60%**
- 8-14 天双 microcycle 逐步减量方案也常见
- 次要比赛：赛前 2-3 天短减量即可（不需要完整 taper）

### 5.2 峰值维持时间

- 从赛前准备到进入 Zone 1（≤2% 低于个人最好成绩）：需要 **4-6 个 microcycle**
- Zone 1 持续时间：**1-2 个月**（内可有 2-3 次真正峰值）
- 神经细胞最佳工作状态持续：**7-10 天**（单次峰值）
- 每次峰值后需 1 个短期再生阶段

### 5.3 峰值分区（Peaking Zones）

| Zone | 表现水平 | 100m 示例（PB 10.00s） | 马拉松示例（PB 2:10:00） |
|---|---|---|---|

| 2 | 2-3.5% | 10.11-10.20 | 2:12:37-2:14:33 |
| 3 | 3.5-5% | 10.21-10.30 | 2:14:34-2:16:30 |
| 4 | >5% | >10.30 | >2:16:30 |

## Section 7：Block Periodization（模块周期化）

### 7.1 Bompa 对 Block Periodization 的定位

**原文（P97/P109）明确界定**：
- "Block" 这个术语不能替代 "Phase"，**除非**是指集中负荷（concentrated load）仅训练一种生物运动能力（力量、速度或耐力），同时牺牲其他能力的情况
- 纯 Block 周期化模型：一般备赛期 = 纯力量块；专项备赛期 = 速度或耐力块（无力量维持训练）
- Bompa 的评估："极少有精英运动员真正以纯 Block 周期化方式训练"

### 7.2 Issurin（2008）的观点 vs Bompa 的反驳

**Issurin 主张**：
- 生物运动能力整合必须是**顺序性的**（sequential），而非同时性的（simultaneous）
- 现代竞技期很长（如职业足球 10-10.5 个月）→ 需要顺序整合
- ATR 模型（Accumulation-Transmutation-Realization）= 积累 → 转化 → 实现，短周期多次重复

**Bompa 的反驳**：
- 周期化本来就隐含了能力发展的**顺序性**（sequencing），但能力整合是**同时性的**（simultaneous integration）
- 一个能力的发展正向影响其他能力（示例：高速度 = 高最大力量 + 高爆发力同时存在）
- 在竞技期仅 3-4 周的足球中，"力量块"期间队伍表现和伤病率会如何？
- ATR = 多峰计划的变体，无法最大化运动员潜能

### 7.3 Block 周期化的实际参照

| 概念 | Bompa 框架中对应 |
|---|---|
| 积累块（Accumulation Block） | AA + Hypertrophy（高 volume，低强度） |
| 转化块（Transmutation Block） | MxS + Conversion（高强度，降 volume） |
| 实现块（Realization Block） | Maintenance + Peaking（减量，维持高质量） |

Bompa 认为这不是新方法，而是原有周期化的**重新命名**。

## Section 8：与 David NCAA 四训和 Joe Kenn Tier 的对应

### 8.1 David NCAA 一周四训（88/68/86/71）强度日解读

David 的四日训练 RPE 模式（推断为：88/68/86/71 → 约重/轻/重/中）：

| 训练日 | 推断 RPE/负荷 | Bompa 对应 |
|---|---|---|
| Day 1（88） | 高强度 | 无氧磷酸系日（力量/爆发力为主） |
| Day 2（68） | 低-中强度 | 有氧或低强度日（超量恢复促进） |
| Day 3（86） | 高强度 | 第二峰值（2-peak microcycle 模式） |
| Day 4（71） | 中等强度 | 技术/战术训练日（恢复类） |

**Bompa 双峰微周期（2-peak microcycle）模式完全匹配**：  
周内两次高需求训练峰值 + 中低需求分隔（图 7.11-7.14 描述）

**对应 Bompa 原则**：
- 高-低-高-中 交替 → 避免疲劳积累
- 避免连续两天极高需求
- 每周 4 训频率符合 "advanced athletes benefit from 4-5 sessions/week" 原则

### 8.2 Bompa 周期化与 Joe Kenn Tier System 对应

**Joe Kenn Tier System**：
- Tier 1 = 多关节力量动作（深蹲/硬拉/奥举）
- Tier 2 = 辅助/单关节动作
- Tier 3 = 爆发力/增强式动作

**Bompa 框架对应**：

| Kenn Tier | Bompa 相位 | Bompa 力量类型 |
|---|---|---|
| Tier 1（AA 阶段） | Phase 1 解剖适应 | 全身多关节，低强度高次数 |
| Tier 1（MxS 阶段） | Phase 3 最大力量 | 高强度（85-90% 1RM），低次数 |
| Tier 1+3 混合 | Phase 4 转化为爆发力 | 中等负荷（40-80% 1RM），爆发速度 |
| Tier 2 全年 | 补偿/解剖适应维持 | 拮抗肌平衡，稳定性 |
| Tier 3（竞技期） | Phase 5 维持 | 维持爆发力，减少 volume |

**具体对应逻辑**：
- Kenn 的 "PCCP"（Power Clean Complex Protocol）= Bompa 的 Cluster Sets for Power（30s 组内休息，3min 组间）
- Kenn 强调"每个运动员每次进场都要有爆发力训练" → 对应 Bompa 的"转化阶段到维持阶段，爆发力训练从不停止"
- Kenn 的竞技期负荷减少（tapering week）= Bompa 的 unloading microcycle（负荷降 40-60%）

### 8.3 NCAA 大学橄榄球年度结构（Bompa 视角）

参考 Bompa P231（文中明确提到 NCAA football）：
- 竞技期 macrocycle 8-9：常规赛；每周以比赛结束 → 单峰 microcycle（峰值在周二或周三）
- Macrocycle 10：联盟冠军赛（Division Championship）
- Macrocycle 11：全国冠军赛或 Bowl Game
- 赛前减量：1 个 unloading 天即可用于小比赛（减量 3-7 天）

**David 的备赛期对应 Bompa 备赛期强度分配**：
- 秋训前 8-12 周（夏训）= Bompa 专项备赛（Specific Preparation）
- AA 阶段在春训 / 春末完成（April-May）
- MxS 在初夏（May-June）
- 转化为爆发力在入秋前（July-August）
- 竞技期维持（September-December）
