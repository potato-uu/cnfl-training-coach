## Section 1：NSCA 核心原则（与其他学派的差异）

### 1.1 竞技能力金字塔：NSCA 对爆发力层级的定义

**核心主张：最大力量是爆发力的基础，不可跳跃。**

- 金字塔从下到上：最大力量 → 爆发力 → 速度 → 敏捷性
- 未经训练者仅靠增大最大力量即可显著提升爆发力（纵跳和立定跳远测量证实）
- 经验越丰富，减少"爆发力赤字（ESD）"越重要：即在有限时间内生成最大力的能力
- 冲刺时脚接触地面时间约 1/100 秒，颈后深蹲髋伸展时间无限长 — 两者 ESD 差距极大

**与 Dietz 三相训练的差异：**  
NSCA 先建最大力量基础，再发展爆发力；Dietz 三相同时在三个收缩相上施加刺激。  
NSCA 方法对新手（1-3年训练年龄）更安全，线性进阶更清晰。

### 1.2 橄榄球能量系统定位

**3秒/20秒法则：**
- 每次攻防持续 **3~6 秒**（取决于战术、人员、场上位置）
- 每次攻防间休息 **20~40 秒**
- 主要供能系统：ATP-PC（"信用卡"）、有氧（"工资"）、无氧糖酵解（"现金"）
- 有氧系统在攻防间隙负责补充 ATP-PC，基础有氧能力越好，恢复越快

**反直觉点：** 不要过度模拟比赛代谢压力（如 2 分钟强制冲刺训练）。这些代谢副产物丰富的环境（H+、K+、Pi）会妨碍适应性，应偶尔暴露但不应作为主要训练手段。

### 1.3 NSCA 专项性原则的边界

- 专项性原则（1945年 DeLorme）：适应取决于训练方法
- **关键限制：** 不应在每次训练中都试图完全模拟比赛需求，那样无法实现超负荷
- 力量房训练要在"同一平面 + 相似爆发力输出 + 模仿比赛姿势"三点上尽量接近比赛，而非完全复制技术动作
- 橄榄球是"以推为主导的运动"（pushing-dominant sport）— 卧推、深蹲、硬拉是基础，不可省略

### 1.4 激活后增强效应（PAP）的标准用法

- PAP：大重量训练前后激活神经肌肉系统，提升随后爆发力输出
- 机制：钙离子分配激活，肌细胞在低刺激频率下生成更大力
- **标准配对：** 大重量深蹲 → 立即跳箱（复合训练）
- 在快缩肌纤维中效果最显著，训练有素者获益更大

### 1.5 颈部/斜方肌训练是橄榄球专有优先级

**有直接证据：** 强化颈部力量可减少脑震荡次数（有身体接触运动的研究支持）
- 更强的颈部 → 碰撞时头部更稳定 → 颅内脑部移动减少 → 受伤减轻
- 推荐训练：奥林匹克举重变式（翻杠抓举）、直接耸肩、4个方向颈部训练器、手动4方向抗阻
- 臂丛损伤（颈肩撞击产生的神经痛，手臂刺痛+无力）是 TE/FB/LB/DB 特有风险，需专项预防

## Section 2：位置专项训练处方

### 2.1 位置分类体系

NSCA 将 11 个位置归为 **6 大类**，训练处方按组别区分：

| 位置组 | 包含位置 | 核心需求 |
|--------|---------|---------|
| 进攻线锋/防守线锋 | OT/OG/C/DT/DE | 最大力量第一，短距离爆发力 |
| 近端锋/全卫/线卫 | TE/FB/LB | 力量+速度均衡，臂丛损伤风险 |
| 外接手/跑卫 | WR/RB | 相对力量 > 绝对力量，速度优先 |
| 防守后卫 | CB/SS/FS | 多方向速度，最大覆盖范围 |
| 四分卫 | QB | 肩部柔韧+稳定，旋转爆发力 |
| 踢球手/弃踢手 | K/P | 髋部主导，单侧对称性 |

### 2.2 进攻线锋（OL）/ 防守线锋（DL）

**生理特征：**
- 需要承受并传递大碰撞负荷，最大力量需求全场最高
- 以屈膝前倾/蹲伏姿势开始每次攻防（三点式站位/直立站位）
- 进攻线锋：几乎每轮都需要持续推压或阻挡，体能需求极高
- 防守线锋：需要侧滑步、踢步、游动、上摆臂过人等侧向动作

**核心受伤部位：** 膝、肩、手、脚踝（进攻线锋）；膝盖、脚踝（防守线锋，体重更大）

**训练处方差异：**
- 核心动作：直杠颈后深蹲（不可替代）+ 直杠硬拉 + 卧推 + 奥林匹克举重
- 阶段2/3添加：**卧推锁定**（强化近线最后推压）、**铁链颈后深蹲**（动态强度，增强爆发力）、**六角杠铃硬拉**（增强站姿发力）
- 阶段3：药球胸前传球（重量递减，增强爆发力）；颈后深蹲后叠加跳箱（PAP）
- 赛季中训练2：继续颈后深蹲（不替换为单腿深蹲），因该位置身体对抗最激烈

### 2.3 近端锋（TE）/ 全卫（FB）/ 线卫（LB）

**生理特征：**
- TE：需要跑传球路线 + 阻挡，冲刺多于线锋但少于技巧性位置
- FB：重心低，下肢力量全队最强，但绝对速度不突出
- LB：承受冲击力可能超过线锋，同时需要速度和覆盖传球路线能力
- 碰撞频率高 → **臂丛损伤高发位置**

**专项防护（臂丛损伤预防协议）：**
- 等距颈部拉伸：每个方向 2组×5次（各个赛季阶段）
- 臂丛损伤组合：每个方向 2组×10次
- 哑铃颈部拉伸耸肩：每侧 2组×10次

**训练处方差异：**
- 硬拉用**六角杠铃**代替直杠（更安全，减少腰椎压力）
- 用**悬垂高翻**代替地面高翻（减少学习门槛）
- 增加单腿练习（弓箭步行走、单腿深蹲），保留 2 组（线锋不做单腿练习）
- 铁链颈后深蹲 + 跳箱复合（增强爆发力），同时保留单腿组

### 2.4 外接手（WR）/ 跑卫（RB）

**生理特征：**
- WR：承受力量负荷全场最大（高速冲刺+切入+碰撞），跟腱、腘绳肌、小腿易伤
- RB：下肢力量强，需要突破截锋 + 保持低重心；承受高速冲击，脑震荡/ACL/肩脱臼风险高
- 不想增加体重影响速度

**训练处方差异：**
1. 六角杠铃硬拉代替直杠
2. **强调单腿练习**（弓箭步行走、踏步登阶、单腿深蹲、单腿腘绳肌练习）
3. 悬垂高翻/悬垂高抓替换地面版本
4. 哑铃代替杠铃做推举练习
5. 阶段1/3：**负重引体向上** 4×10→5×5（增加上肢质量但不增加多余体重）
6. **负重反向划船**（Inverted Row）4×10→5×5（代替传统背部划船）
7. WR 特别注意：赛季中需减少下肢大训练量，尤其训练量最重时期

**关键点（离心力量）：** WR 必须增强离心力量（特别是保护性的），同时完成高速后链动作（壶铃甩摆、举重衍生训练、药球弹震式训练）。

### 2.5 防守后卫（DB：CB/SS/FS）

**生理特征：**
- 动作模式最复杂：向后躲闪、侧滑步、后撤步、开放步、45/90/180度快速转向
- 覆盖范围全场最大，高速碰撞时冲击力远大于其他位置
- 向后跑时推进力主要由**膝伸肌**和**髋屈肌**生成（与向前跑不同）

**训练处方差异：**（= TE/FB/LB + WR/RB 的组合）
1. 六角杠铃硬拉代替直杠
2. 强调单腿练习
3. 悬垂高翻/高抓
4. 哑铃推举
5. 负重引体向上/负重反向划船（轮替）
6. 臂丛损伤预防协议（同 TE/FB/LB）

### 2.6 四分卫（QB）

**生理特征：**
- 体形多样，不应期望所有 QB 有相同身体素质（口袋型 vs 双重威胁型）
- 核心需求：上肢力量、肩部灵活性+稳定性、躯干稳定性、下肢力量
- **关键风险：胸肌相对上背部/后三角肌过度发达**（会造成肩部失衡和受伤）

**训练处方差异（全部要点）：**
1. **颈后深蹲用更轻负荷或其他变式**
2. **强调单腿练习**
3. 推举全部改为**哑铃**（减少肩关节压力）
4. 奥林匹克举重用哑铃
5. 硬拉用较轻负荷或六角杠铃/哑铃（台上）
6. **旋转练习**：半跪式上劈/下拉（每侧各方向）
7. **肩袖强化**：外旋 + 使用角形肩部训练器 + 拉力器肩外旋
8. 所有肩部推举使用较轻负荷
9. **握力强化**：等距抓握橄榄球（15秒）+ 伸肌训练带（每方向 2组×10次）
10. **投掷爆发力**：头顶单臂快速投药球 + 减速投药球（都是必须的）
11. 跪姿药球 BLOB 投掷（重量递减）
12. 取消药球胸前传球（不需要这个方向的爆发力）

### 2.7 踢球手（K）/ 弃踢手（P）

**生理特征：**
- 高度单侧主导，摆动腿与支撑腿存在力量/爆发力/灵活性不对称
- 髋屈肌和股四头肌主导

**训练处方差异：**
1. 降低杠铃深蹲强度
2. **重视单腿练习**（分腿蹲、弓箭步、罗马尼亚单腿硬拉、臀桥、单腿跳箱）
3. **4方向臀肌训练器**替代标准腹股沟练习（髋屈/伸/内收/外展）
4. 髋关节外展/内收练习
5. 髋关节屈伸练习
6. 躯干旋转练习
7. 取消药球胸前传球
8. **单腿软着陆训练**：从箱子上落地单脚接触，双脚跳→单脚落地，培养本体感觉

## Section 3：测试方案与位置基准数据

### 3.1 标准测试电池

NSCA 推荐的完整测试组合：

| 测试项目 | 目的 | 重要细节 |
|---------|------|---------|
| 1RM 卧推 | 上肢最大力量 | 3~5次尝试确定1RM；需保护员 |
| 1RM 颈后深蹲 | 下肢最大力量 | 先做医学检查；允许腰带 |
| 1RM 高翻 | 全身爆发力 | 无需保护员；变式多 |
| 纵跳 | 下肢爆发力 | 4种测量法（跳垫/Vertec/测力台/粉笔墙） |
| 立定跳远 | 水平爆发力+减速能力 | 3次取最佳 |
| 40码冲刺 | 直线速度 | 同时记录 20码（加速）和 40码（最高速）|
| 5-10-5 敏捷测试 | 横向速度+变向 | 三点式起跑；两方向各测；3~5分休息 |
| L形敏捷测试（三锥） | 变向能力 | 右手触线；可做反应版本 |
| 300码往返跑 | 无氧能力 | 25码×6往返×2组；组间休息5分钟 |
| 60-50-40码测试 | 位置无氧能力 | 位置专项距离；10次×2组；30秒组间休息 |
| 225磅卧推最大次数 | 上肢肌耐力 | NFL Combine标准；高中可降到185磅 |

### 3.2 测试时机与风险管理

- **最佳时机：** 运动员能熟练完成测试动作时（不是刚开始训练）
- **替代方案（高风险运动员）：** 3~5RM测试代替1RM，10码冲刺代替40码
- 每次测试必须用完全相同方式进行（可靠性）
- 间隔3~5分钟休息（速度和敏捷测试之间）

### 3.3 60-50-40码测试 — 位置专项无氧标准

**最具橄榄球专项性的无氧能力测试：**

| 位置 | 距离 | 时间限制 |
|------|------|---------|
| OL / DL | 40码（约37米）| 6秒内 |
| LB / RB / QB / TE / K / P | 50码（约46米）| 7秒内 |
| WR / DB | 60码（约55米）| 8秒内 |

- 每次重复之间休息 **30秒**
- 每位置完成 **10次重复**
- 共 2 组（组间 3 分钟休息）
- 两组全部达标才算通过

### 3.4 NFL Combine 基准数据（参考等级）

以下为 NFL 选秀考察训练营的分类基准（70%=高，50%=典型，30%=低）：

**纵跳（NFL Combine）：**
- 高（70%）：约 35+ 英寸（约89厘米以上）
- 典型（50%）：约 31~35 英寸（约79~89厘米）
- 低（30%）：约 28~30 英寸（约71~76厘米）

**立定跳远（NFL Combine）：**
- 高（70%）：约 130+ 英寸（约330厘米以上）
- 典型（50%）：约 120~130 英寸（约305~330厘米）

**40码冲刺（NFL Combine）：**
- 快（70%）：<4.5秒（技巧性位置）/ <5.0秒（线锋）
- 典型（50%）：约 4.5~4.7秒（技巧性）/ 5.0~5.3秒（线锋）

**5-10-5敏捷测试（NFL Combine）：**
- 快（70%）：<4.1秒
- 典型（50%）：约 4.1~4.4秒
- 慢（30%）：>4.4秒

**L形敏捷测试（NFL Combine）：**
- 快（70%）：<6.7秒
- 典型（50%）：约 6.7~7.2秒

**225磅卧推最大次数（NFL Combine，各位置）：**
- OL（进攻线锋）：25~35次
- DL（防守线锋）：25~35次
- LB（线卫）：20~30次
- TE（近端锋）：20~30次
- DB（防守后卫）：15~25次
- WR（外接手）：10~20次
- QB（四分卫）：10~20次
- RB（跑卫）：15~25次

### 3.5 大学橄榄球测试规范（NCAA）

| 位置 | 参考成绩（秒/25码） |

## Section 4：年度周期化模板（NSCA版）

### 4.1 年度训练结构概览

```
赛季后（8周）→ 休赛期（11周）→ 赛季前（6周）→ 训练营（4周）→ 赛季中（15周）
```

| 赛季阶段 | 时长 | 主要目标 | 训练重点 |
|---------|------|---------|---------|
| 赛季后 | 8周 | 休息→恢复→基础 | 自重+基础力量，3天/周全身 |
| 休赛期 | 11周 | 最大增益期 | 4天/周分化（上/下肢），3阶段线性 |
| 赛季前 | 6周 | 过渡到比赛状态 | 先4天后3天，2阶段 |
| 训练营 | 4周 | 维持+体能峰值 | 3天/周全身，低训练量 |
| 赛季中 | 15周 | 维持全年训练成果 | 3天/周（可降2天），3周周期 |

### 4.2 休赛期三阶段线性模型

**训练结构：** 每周4天，周一/四上肢，周二/五下肢，周三休息

| 阶段 | 名称 | 负荷范围 | 重复次数 | 目标 |
|------|------|---------|---------|------|
| 阶段1 | 肌肥大/力量耐力 | 50%~85% 1RM | 6~20次 | 增肌 + 提升耐力 |
| 阶段2 | 基础力量 | 80%~95% 1RM | 2~6次 | 最大力量提升 |
| 阶段3 | 力量/爆发力 | 87%~95% (力量) / 30%~85% (爆发力) | ≤5次 | 最大力量+爆发力并行 |

**关键数字：**
- 爆发力最大化卧推负荷：**50%~70% 1RM**（不是大重量！）
- 最大力量组间休息：**3~5分钟**
- 辅助练习（二头肌弯举等）组间休息：**30~90秒**
- 肌肉耐力练习休息：**≤30秒**
- 减量周：每3~4个训练周安排一个（训练营前一周必须减量）

**练习顺序（每次训练课固定顺序）：**
1. 奥林匹克举重（爆发力练习，疲劳前优先）
2. 非爆发力核心练习（深蹲/卧推/硬拉）
3. 辅助练习（孤立肌群）
4. 解剖学核心区域练习（作为结束练习）

### 4.3 赛季前两阶段模型

**结构：**
- 阶段1（基础力量）：4天/周分化（类似休赛期阶段2）
- 阶段2（力量/爆发力）：转换为3天/周全身训练，强调体能

**强度区间：**
- 阶段1：奥林匹克举重 75%~82% 1RM × 3次；核心练习 75%~85% 1RM × 5次
- 阶段2：奥林匹克举重 72%~82% 1RM × 2次（强调速度）；核心练习 80%~92% 1RM × 3次
- 铁链卧推/深蹲：以 50% 1RM + 铁链重量，3~4次重复（动态强度法）

**3:1 周期比：** 3个训练周 + 1个减量周（6周内允许5训1减）

### 4.4 赛季中维持模型

**训练营（4周）：**
- 4个不同全身训练计划轮换，每周2~3次
- 下肢：40%~57% 1RM，3次×4组（极保守，因场上练习量大）
- 上肢：62%~80% 1RM，3组，2~5次（比下肢积极）
- 每周应至少2次，最多3次

**赛季中（15周）：**
- 每周3天（第1天下肢，第2天全身+爆发力，第3天上肢）
- 3周为一个单位，结束后1周减量/变更（不同练习，变化刺激）
- 强度比阶段2降低约15%，重复次数降低约10%
- 辅助练习：8~10次/组
- 可降为每周2天（省去上肢日）

**训练日安排逻辑：**
- 下肢训练日：比赛结束后尽快安排（比赛次日或休息日后）
- 全身爆发力日：休息日后，或下肢日次日
- 上肢日：下次比赛前两天

### 4.5 赛季后两阶段模型

**常规准备阶段（3周，3天/周全身）：**
- 以自重练习为主：俯卧撑、引体向上、平板支撑、开合跳、深蹲、弓箭步
- 目标：动作质量 > 训练量；全活动范围
- 第1周：每个循环至少2次，无休息
- 后续：可增加到3~4次循环；负重背心（从3~5kg开始）

**基础力量阶段（5周，4天/周分化）：**
- 周一/四上肢，周二/五下肢，周三休息
- 逐周递增训练量和强度
- 引入爆发力练习：悬垂高翻高拉、杠铃抓举、跳箱

## Section 5：动作选择与技术要点

### 5.1 四大核心练习体系

NSCA 确立了**四大必练类别**（所有位置）：

| 类别 | 首选动作 | 备选（受伤/限制时） |
|------|---------|------------------|
| **推** | 卧推/上斜卧推、借力推、肩部推举 | 哑铃推举、高位卧推、T形杠铃 |
| **深蹲** | 颈后深蹲、颈前深蹲、安全杆深蹲 | 六角杠铃深蹲、腰带深蹲、哑铃相扑深蹲 |
| **硬拉** | 传统硬拉 | 六角杠铃硬拉、各类硬拉机器 |
| **奥林匹克举重** | 高翻、高抓、借力挺举、组合练习 | 哑铃奥林匹克举重、跳跃训练器、药球抛掷、T形杠铃深蹲转推举 |

### 5.2 奥林匹克举重变式选用逻辑

- **地面高翻/高抓：** 休赛期阶段2/3核心（最大爆发力）
- **悬垂高翻/高抓：** 技巧性位置首选（减少背部压力）；TE/FB/LB 等中间型位置也适用
- **肌肉抓举（Muscle Snatch）：** 阶段1使用，强化髋膝完全伸展能力
- **抓举拉/高翻拉：** 赛季中首选（降低肘腕压力，全年使用）
- **借力挺举（Push Press）：** 极好的髋→腿→手臂力量传递练习（橄榄球以推为主导的本质）
- **挺举（Clean and Jerk）：** 高翻+借力挺举的组合，高效利用时间

### 5.3 单腿练习的战略作用

**适用位置：** TE/FB/LB/WR/RB/DB/QB/K/P（除纯线锋外）

关键动作列表：
- 弓箭步行走（Walking Lunge）
- 踏步登阶（Step-Up）
- 单腿深蹲（Single-Leg Squat）
- 分腿蹲（Split Squat）
- 反向弓箭步（Reverse Lunge）
- 箭步蹲（Lunge）
- 罗马尼亚单腿硬拉（Single-Leg Romanian Deadlift）

**逻辑：** 橄榄球中大量单腿蹬地、切入、跳跃、落地动作 → 必须有单腿专项力量。

### 5.4 铁链训练的作用（动态强度法）

- 铁链颈后深蹲：阶段2/3出现，赛季前和赛季中减量周使用
- 铁链卧推：同上
- **原理：** 杠铃重量减轻，但必须快速移动 → 同时增强爆发力
- **读法标注：** 50CH/3×4 = 50%1RM + 铁链，4组，每组3次

### 5.5 药球练习的具体编排

**胸部/上肢爆发力：**
- 药球胸前传球（站姿 + 启动姿势）：OL/DL 专用，站姿3次+启动模式3次，重量递减
- 药球落下接球（Drop and Catch）：增强反应式接球能力
- 单臂头顶投掷（快速）：QB 投掷爆发力
- 单臂头顶减速投掷：QB 手臂减速训练

**旋转/核心爆发力：**
- 坐姿伙伴侧抛：躯干旋转能力
- 侧旋过臀抛药球
- 环绕式药球练习（重量递减）
- 药球上劈/下拉：QB 投掷爆发力

### 5.6 后链肌群的橄榄球专项地位

- 现代橄榄球越来越注重速度 → 训练重点转向后链（臀大肌、腘绳肌、下背部）
- 腘绳肌拉伤是最常见肌肉伤病（冲刺时）→ 根源是股四头肌/腘绳肌失衡
- **推荐后链动作：**
  - 臀大肌-腘绳肌伸展（GHD）
  - 反向背部伸展
  - 罗马尼亚硬拉
  - 臀推（Barbell Hip Thrust）/ 单腿臀桥
  - 壶铃甩摆（Kettlebell Swing）—— 赛季前重点练习之一

## Section 6：康复与伤病预防

### 6.1 常见橄榄球伤病分布

| 伤病类型 | 高风险位置 | 预防重点 |
|---------|----------|---------|
| 腘绳肌拉伤 | WR/RB/DB（冲刺时） | 后链力量均衡，腘绳肌专项训练 |
| ACL 撕裂 | 所有技巧性位置 | 减少膝外翻，正确落地技术 |
| 脑震荡 | 所有位置 | 颈部力量训练（4方向） |
| 臂丛损伤 | TE/FB/LB/DB | 专项颈肩预防协议 |
| 踝关节扭伤 | OL/技巧性位置 | 踝关节稳定性训练 |
| 肩关节损伤 | QB（投掷）/WR（碰撞）| 肩外旋强化，前后肌肉平衡 |
| 膝/脚踝损伤 | OL（体重大）| 下肢灵活性+力量专项 |

### 6.2 ACL 预防协议

两个关键干预：
1. **减少屈曲时过度膝外翻**（训练中严格纠正）
2. **指导正确落地技术**（软着陆，避免直膝落地）

### 6.3 颈部强化协议

**直接颈部训练（4方向）：**
- 颈部训练器（4方向器械）
- 手动4方向抗阻训练（需训练伙伴）
- 等距颈部拉伸（每方向 2组×5次）

**间接颈部训练（通过奥林匹克举重激活斜方肌）：**
- 翻杠抓举变式
- 哑铃/杠铃耸肩（爆发力耸肩）

### 6.4 肩部前后肌肉平衡

**原则：** 推举训练导致内旋肌肉强化，必须加入外旋训练平衡
- 肩外旋练习（哑铃/拉力器）
- 使用角形肩部训练器
- 背阔肌下拉 / 俯身划船（建立推/拉平衡）

**QB 特别提醒：** 避免胸肌远超背部/后三角肌，否则肩关节健康风险极高。

### 6.5 踝关节稳定性训练

- 平衡踝关节稳定训练（贯穿休赛期/赛季前/赛季中）
- 单脚软着陆训练（特别适用于 K/P）
- 提踵走（改善灵活性、协调性和稳定性）

### 6.6 赛季后恢复原则

- 赛季结束后**至少休息 1 周**（被动恢复）
- 然后进入常规准备阶段（自重练习，注重动作质量）
- 监控体重：体脂百分比和瘦体重应与赛季中保持接近（允许小幅波动）
- 瘦体重减少 + 体脂增加是最大风险，应防范

## Section 7：特殊人群（如书中涉及）

本书主要面向**高中、大学、职业**三个级别，未设独立青少年/女性章节。  
但以下内容对 CNFL 业余选手有直接参考价值：

### 7.1 高中生/新手运动员的处理原则

- **不适合一开始就做1RM测试**：动作质量不达标时先用3~5RM代替
- 未经训练的运动员只靠增大最大力量即可显著提升爆发力（跳跃/短距离冲刺）
- 225磅卧推最大次数测试：高中球员建议降到**185磅**（更合适的测试重量）
- 早期专业化的年轻运动员容易出现全年参赛 → 过度疲劳风险，需要更多休息

### 7.2 实习组球员 / 红衫球员（适用于 CNFL 替补或轻训选手）

赛季中如不参加比赛，可使用**改版的休赛期训练计划**，目标是提高而非维持。
