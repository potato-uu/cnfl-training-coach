# Profiles — 本地私有运动员档案

此目录被 `.gitignore` 屏蔽，不会提交到 repo。每位使用者在本地存放自己的**个人缺陷专项修复协议**。

## 为什么私有

档案内含：体重、1RM 数据、速度测试、伤病史、专项处方。属于个人健康与训练数据，不应公开。

## 文件命名

`knowledge-[athlete-id]-protocols.md`

示例：`knowledge-stan-protocols.md` / `knowledge-mike-protocols.md`

## 教练如何使用

SKILL.md 的路由条目指向 `profiles/knowledge-[athlete-id]-protocols.md`。
当教练识别到运动员 id（通过姓名/代号字段匹配），会优先读取该档案而非通用处方。
档案不存在时，教练回退到通用处方 + 提示"建议建立个人协议档案"。
